## 📦 What is ai-dev-kit?

ai-dev-kit is a toolkit designed to help you work with Databricks coding agents. It includes tools built by Field Engineering to make coding with agents like Claude and Cursor easier. You do not need any programming skills to use this software. The tools provide a simple way to interact with Databricks and coding agents on your Windows computer.
